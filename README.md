# ☕ Yewale Amruttulya Tea Sales Analysis

## Project Overview

This project analyzes tea sales data using SQL, Python, and Power BI to discover business insights.

## Objectives

- Analyze monthly sales
- Identify top-selling products
- Find high-performing branches
- Compare revenue across regions
- Visualize KPIs

## Tools Used

- Python
- Pandas
- NumPy
- SQL
- Power BI
- Excel

## Dataset

The dataset contains:
- Date
- Branch
- Product
- Quantity
- Revenue
- Profit

## Project Workflow
